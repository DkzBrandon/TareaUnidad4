Tarea: Conceptos y ejemplos básicos en HTML

¿Qué es? Técnicas utilizadas para proteger los formularios de ataques como inyecciones SQL o XSS.

¿Para qué se usa? Garantizar que los datos enviados sean válidos y seguros.

¿Cuándo se usa? En formularios donde se recolecten datos de los usuarios, como inicios de sesión o registros.

                    
                        <form method="POST" action="/submit">
                            <input type="text" name="username" required>
                            <input type="hidden" name="csrf_token" value="random_generated_token">
                            <button type="submit">Enviar</button>
                        </form>

Resultado esperado: Se realiza una solicitud HTTP GET a https://api.example.com/data. Si la respuesta tiene un código de estado 200, el contenido de la respuesta se muestra en la consola del navegador.

¿Qué es? Técnica para enviar y recibir datos sin recargar la página completa.

¿Para qué se usa? Hacer aplicaciones web más rápidas y dinámicas.

¿Cuándo se usa? En búsquedas en vivo, autocompletados y formularios avanzados.


    let xhr = new XMLHttpRequest();
xhr.open("GET", "https://api.example.com/data", true);
xhr.onload = function () {
    if (xhr.status === 200) {
        console.log(xhr.responseText);
    }
};
xhr.send();

Resultado esperado: Al hacer clic en el botón con ID #button, el texto dentro del elemento con ID #text se cambiará a "Texto cambiado".

¿Qué es? Una librería de JavaScript que simplifica tareas comunes como manipulación de elementos y animaciones.

¿Para qué se usa? Reducir el tiempo necesario para escribir código repetitivo.

¿Cuándo se usa? Para manejar eventos, manipular el DOM o agregar efectos visuales fácilmente.

$(document).ready(function () {
    $("#button").click(function () {
        $("#text").text("Texto cambiado");
    });
});

Resultado esperado: Al hacer clic en el botón, el texto del elemento cambiará a "¡Texto cambiado!".

¿Qué es? Un framework de JavaScript que facilita la creación de interfaces avanzadas y efectos visuales.

¿Para qué se usa? Agregar animaciones, manipular elementos del DOM y mejorar la experiencia del usuario.

¿Cuándo se usa? En proyectos donde se requiere funcionalidad avanzada en interfaces.

¿Qué es? Una librería de JavaScript que facilita tareas como peticiones AJAX y manipulación del DOM.
